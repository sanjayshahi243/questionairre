API Overview
Core Endpoints
  • /api/question-sets/ - Manage question sets and versions
  • /api/questions/ - Manage reusable questions
  • /api/question-set-questions/ - Manage questions within sets
  • /api/answers/ - Manage user answers
  • /api/dependencies/ - Manage conditional logic
Authentication

Most endpoints require authentication. You can use:

  • Token Authentication: Include Authorization: Token your-token header
  • Session Authentication: For browser-based access
Getting Started
  1. Create a question set using POST /api/question-sets/
  2. Add questions to the set using POST /api/question-set-questions/
  3. Set up dependencies using POST /api/dependencies/
  4. Collect answers using POST /api/answers/
